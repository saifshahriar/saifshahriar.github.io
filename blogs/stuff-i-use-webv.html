Intro

I have a kind of a minimal setup. I am biased and I mostly use what works for me. I mostly try to adhere to the UNIX Philosophy: Do one thing and do it right. Also, influenced by the Suckless Philosophy. So, I am basically DOOMED! hehe.

Setups

  • Primary OS: Void Linux (Previously using Artix, Arch & OpenBSD)
  • Secondary OS(s): Plan9 - Such an elegant piece of art.
  • Window Manager: Dynamic Window Manager - DWM
  • Terminal Emulator: Simple Terminal - st (Luke Smith’s Build)

GUI Programs

  • Browser: Brave Browser
  • File Manager: PCManFM
  • Image Viewer: Sxiv
  • Video & Audio: mpv
  • Presentation: sent

Terminal Programs

  • IDE/Editor: (Neo)Vim
  • File Manager: Yazi

Tools

  • Dmenu - A suckless menu literally has the power to be anything.

  • Slock - A suckless screenlock.

  • Xmenu - A window manager agnostic suckless context menu.

  • farbfeld - A lossless image format.

  • LaTeX - A bloated document preparation system. Use groff(1) instead.

  • groff - Front-end for the groff(1) document formatting system.

  • pandoc - A general markup converter.

  • fish - Friendly Interactive Shell.

  • eza - A better ls(1) & successor of exa.

  • bat - A better cat(1).

  • delta - A better diff(1).
